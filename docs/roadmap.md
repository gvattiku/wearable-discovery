# Roadmap

## Phase 1
- Finalize product positioning and messaging.
- Build a polished landing page.
- Invite people to share lived experience for early research.

## Phase 2
- Refine the sensing concept and user experience.
- Run lightweight user testing.
- Define the MVP feature set.

## Phase 3
- Prepare for prototype development.
- Explore hardware and companion app direction.
- Evaluate early pilot opportunities.
